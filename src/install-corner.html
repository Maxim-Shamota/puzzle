Немного о душевом уголке

Самый популярный душевой уголок — 90×90. Этот размер является наиболее удобным для установки в небольшой ванне и позволяет комфортно принимать душ.

Однако, если дизайн решение позволяет, можно использовать и другие размеры душевого уголка 80×80, 100×100, 120×80 и другие. Если вы планируете установить душевой уголок с поддоном, звоните в «Боб-мастер»!

Наши специалисты имеют многолетний опыт работы по установке душевых уголков с низким, высоким поддоном и установленным подиумом.

1Разметка для установки душевого уголка
